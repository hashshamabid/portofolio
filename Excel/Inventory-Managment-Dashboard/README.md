# Inventory Management Dashboard

## Overview
An Excel-based inventory management dashboard developed to monitor stock levels, reorder requirements, supplier lead times and inventory costs.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Data Validation
- Dashboard Design

## Dashboard Features
- Total SKU tracking
- Low stock monitoring
- Out-of-stock analysis
- Reorder point tracking
- Average supplier lead time
- Total reorder cost calculation
- Inventory value analysis
- Stock movement insights

- KPI reporting
- Business data analysis
- Data visualisation
