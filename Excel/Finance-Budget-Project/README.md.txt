## Finance Budget Planner Dashboard

### Business Objective
To track income, expenses, net profit, and available cash while enabling
scenario-based expense adjustments for financial planning.

### Dataset
Monthly financial data including:
- Income
- Expense categories (Salary, Rent, Utilities, Marketing, etc.)
- Net Profit
- Available Balance

### Key KPIs
- Total Income
- Total Expenses (Actual vs Adjusted)
- Net Profit (Actual vs Adjusted)
- Available Balance
- Rolling 3-Month Average

### Key Features
- Scenario control for expense adjustments
- Rolling average calculations
- Automated net profit and balance tracking
- Interactive financial dashboard

### Tools & Techniques
- Advanced Excel formulas
- Pivot Tables
- Scenario analysis
- Dashboard design best practices

### Outcome
The model enables better budgeting decisions by showing the impact of cost changes
on profitability and cash flow.
