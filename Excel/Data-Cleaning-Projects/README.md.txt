## Data Cleaning Project – Manual & Power Query

### Business Objective
To clean raw customer transaction data and prepare it for reliable reporting
and analysis.

### Raw Data Issues
- Duplicate records
- Inconsistent customer names
- Invalid and inconsistent email formats
- Formatting inconsistencies
- Missing values

### Cleaning Approaches
#### 1. Manual Cleaning
- Removed duplicates
- Standardized customer names
- Corrected email formats
- Fixed date and currency formats

#### 2. Power Query Cleaning
- Automated duplicate removal
- Text standardization
- Data type corrections
- Reusable and refreshable cleaning steps

### Final Output
- Clean transaction dataset
- Customer-level summary table
- Total orders and total order value per customer

### Tools & Techniques
- Excel formulas
- Power Query (Transform, Clean, Merge)
- Data validation

### Outcome
This project demonstrates the ability to clean messy data efficiently
and prepare high-quality datasets for analysis.
