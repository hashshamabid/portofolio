# Data Cleaning Project

## Overview
This project demonstrates how messy business data can be cleaned, standardised and prepared for analysis using Microsoft Excel and Power Query.

## Project Objectives
- Improve data quality
- Remove inconsistencies
- Standardise formats
- Prepare data for reporting and analysis

## Issues Identified
- Inconsistent capitalisation in names
- Extra spaces in customer names
- Currency values stored as text
- Dates stored as text
- Duplicate customer records
- Invalid email formats
- Inconsistent customer IDs
- Blank and malformed rows

## Cleaning Tasks Performed
- Removed duplicate records
- Standardised customer names
- Converted text values to numeric values
- Corrected date formatting
- Fixed email formatting issues
- Standardised customer IDs
- Removed unnecessary spaces
- Validated data quality

- Microsoft Excel

- Data Validation
- Conditional Formatting
