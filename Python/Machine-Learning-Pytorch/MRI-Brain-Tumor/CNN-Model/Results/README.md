# Results

This folder contains the evaluation results of the custom CNN model.

## Contents

- Training Loss Curves
- Validation Loss Curves
- Training Accuracy Curves
- Validation Accuracy Curves
- Confusion Matrix
- Classification Report

## Purpose

These results are used to evaluate the performance of the CNN model on the Brain Tumor MRI dataset.

## Metrics

- Accuracy
- Precision
- Recall
- F1-Score

## Notes

The visualizations and metrics help identify model strengths, weaknesses, and classification performance across different tumor classes.
