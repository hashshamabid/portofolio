# Custom CNN Model

This folder contains a Convolutional Neural Network (CNN) developed from scratch using PyTorch for Brain Tumor MRI classification.

## Model Architecture

- Convolutional Layers
- ReLU Activation
- Max Pooling
- Fully Connected Layers
- Softmax Classification

## Purpose

The model was trained to classify MRI brain scan images into different tumor categories.

## Technologies Used

- Python
- PyTorch
- NumPy
- Matplotlib
- Google Colab

## Notes

This model was developed as part of a deep learning project to compare custom CNN performance against pre-trained models 
such as DenseNet, EfficientNet, ResNet50, and VGG16.
