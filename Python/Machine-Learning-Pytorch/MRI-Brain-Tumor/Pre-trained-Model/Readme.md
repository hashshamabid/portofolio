# Brain Tumor Classification Using Pre-trained Models

This project compares multiple transfer learning architectures for brain tumor MRI classification.

## Models Evaluated

- ResNet50
- VGG16
- EfficientNet-B0

## Dataset

Brain Tumor MRI Dataset

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

## Results

The results folder contains training curves, confusion matrices, and performance metrics for each model.
